# archive/ — old miscellaneous files, not part of the running app

Nothing here is used by `tracker/app.py`. It's kept instead of deleted in
case any of it is ever worth a second look. (Cloud/Streamlit-Cloud-specific
material lives separately in `cloud-archive/` at the repo root — this
folder is just leftover odds and ends.)

| File/folder | What it is | Why it's here |
|---|---|---|
| `app.py.bak_20260714_134832` | A full backup of `tracker/app.py` from partway through development (2026-07-14, before most current features existed) | Point-in-time snapshot in case anything from that stage is worth diffing against. Very stale — not a reference for current behavior. |
| `homeschool-parks-app.textClipping` | A macOS Finder "text clipping" file, created by dragging selected text onto the Desktop | Landed in the project folder by accident; not real project content. Kept in case it held a note that mattered. |
| `calendar/weekly-schedule.html` | A standalone static HTML weekly-schedule page, built independently of the Streamlit app | Predates the app's own "My Week" tab, which now does this natively with live data. Not linked from anywhere in the app. |
